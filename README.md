# araBeat
An interactive New Media Art system to translate the heartbeat of the visitor to an artistic form of sound and light in real-time.

This is a 12-feet-tall-permanent-public-installation, therefore it is built to withstand a harsh environment where the temperature goes to -20Â°C.
